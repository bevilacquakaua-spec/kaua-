# 🐂 Blog - Bovinocultura de Corte

Projeto desenvolvido utilizando HTML, CSS e JavaScript.

## Objetivo

Apresentar informações básicas sobre a bovinocultura de corte, incluindo:

- Introdução
- Principais raças
- Manejo
- Alimentação

## Tecnologias

- HTML5
- CSS3
- JavaScript

## Como executar

1. Baixe ou clone o projeto.
2. Abra o arquivo `index.html` em qualquer navegador.
